# README Media Refresh Checklist

Use this checklist before updating `README.md` screenshots, GIFs, or caption text.

## What to verify

- The screenshot matches the current navigation labels and page names.
- The screenshot matches the current product split:
  - `Findings`
  - `Sources`
  - `Security Graph`
  - `Agent Mesh`
  - `Remediation`
- The screenshot does not show incidental local noise, stale branch state, or unrelated runtime errors.
- The caption describes the surface honestly and does not overstate an unfinished flow.
- The README text still matches the current product story after the screenshot is updated.

## What to refresh together

- `dashboard-live.png`
- `mesh-live.png`
- `remediation-live.png`
- `demo-latest.gif`
- the corresponding README captions

## What not to change here

- Terminal demo scripts and capture automation
- Product code that is unrelated to README/media alignment
- Old screenshots unless the updated capture is ready to replace them
