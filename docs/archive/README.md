# Archived Docs

This folder keeps low-signal or superseded operator docs that are still worth
retaining in the repo but should not crowd the primary documentation surface.

These files are intentionally not treated as the canonical product/deployment docs.
Do not park private strategy memos, persona scorecards, or agent-session audit
ledgers here — those stay in private notes (see `AGENTS.md` and
`scripts/check_public_docs_hygiene.py`).
