  routes with non-empty graph fixtures.

  vulnerabilities, credentials, traversable edges, attack paths, snapshot

## Claim Boundaries

- These screenshots prove the interactive cockpit states and large-graph
  drilldown behavior in browser tests; they are not marketing mockups.
- They do not claim unbounded browser rendering for every graph size. Large
  snapshots intentionally use overview and drilldown modes.
- Published README images should only be replaced through the packaged capture
  flow in `docs/CAPTURE.md`.
